## 1. 현재 구현된 기능 요약

### 스캔 기능
- 카메라 실시간 문서 스캔
- ONNX 기반 딥러닝 문서 감지 (DocAligner)
- OpenCV 기반 문서 영역 감지 (fallback)
- 원근 변환 및 크롭
- 자동 스캔 모드 (문서 감지 시 자동 촬영)
- 연속 촬영 (배치 모드)
- 음성 명령 촬영

### 이미지 처리
- 필터 적용 (원본, 문서 모드)
- 그림자 제거
- 화이트밸런스 보정
- 곡률 보정
- 손가락 감지

### 저장 및 공유
- 이미지 저장 (PNG/JPG)
- PDF 생성
- 공유 기능

### 관리
- 스캔 이력 갤러리

## 2. 기능 확장 기획

### 2.1 OCR 텍스트 인식 기능
**목표**: 스캔한 문서에서 텍스트를 추출하여 복사/검색 가능하게 함

**세부 사항**:
- Google ML Kit 또는 Tesseract OCR 통합
- 한글/영문 텍스트 인식 지원
- 추출된 텍스트 클립보드 복사
- 텍스트 검색 기능
- OCR 결과 편집 기능

**기술 스택**:
- `google_mlkit_text_recognition` 패키지
- 또는 `tesseract_flutter` 패키지

---

### 2.2 클라우드 동기화
**목표**: 스캔한 문서를 클라우드에 백업하고 다른 기기에서 접근 가능하게 함

**세부 사항**:
- Google Drive 연동
- Dropbox 연동 (선택사항)
- 자동 백업 옵션
- 동기화 상태 표시
- 오프라인 모드 지원

**기술 스택**:
- `googleapis` 패키지
- `flutter_dropzone` 또는 유사 패키지

---

### 2.3 폴더/태그 관리 시스템
**목표**: 스캔한 문서를 체계적으로 정리할 수 있게 함

**세부 사항**:
- 사용자 정의 폴더 생성
- 태그 시스템 (색상 태그)
- 즐겨찾기 기능
- 검색 기능 (파일명, 태그)
- 정렬 옵션 (날짜, 이름, 크기)

**UI 구성**:
```
홈 화면
├── 전체 문서
├── 즐겨찾기
├── 폴더 목록
│   ├── 업무
│   ├── 개인
│   └── 영수증
└── 태그 필터
```

---

### 2.4 문서 편집 기능 강화
**목표**: 스캔 후 추가 편집 기능 제공

**세부 사항**:
- 회전/반전 기능
- 밝기/대비/채도 조절
- 자르기 기능
- 워터마크 추가
- 서명 추가 기능
- 텍스트 주석 추가

---

### 2.5 다중 페이지 PDF 편집
**목표**: PDF 생성 시 페이지 순서 변경 및 관리

**세부 사항**:
- 드래그 앤 드롭으로 페이지 순서 변경
- 페이지 삭제/추가
- 페이지 미리보기
- PDF 병합 기능
- PDF 분할 기능

---

### 2.6 자동 문서 분류
**목표**: AI를 활용한 문서 자동 분류

**세부 사항**:
- 문서 타입 자동 인식 (영수증, 명함, 계약서 등)
- 자동 폴더 정렬 제안
- 날짜 자동 추출 (영수증, 계약서 등)

## 3. 사용자 경험 개선 기획

### 3.1 온보딩/튜토리얼
**목표**: 신규 사용자가 앱 사용법을 쉽게 익히도록 함

**세부 사항**:
- 첫 실행 시 가이드 투어
- 기능별 툴팁
- 스캔 가이드 오버레이 (문서를 어떻게 배치해야 하는지)
- 자동 스캔 모드 설명

**구현 방법**:
- `introduction_screen` 패키지 활용
- `tutorial_coach_mark` 패키지 활용

---

### 3.2 스캔 UX 개선

#### 3.2.1 실시간 가이드 오버레이
**문제**: 사용자가 문서를 올바르게 배치했는지 알기 어려움

**개선안**:
- 문서 정렬 가이드라인 표시
- "문서를 더 가까이 가져오세요" 등 실시간 피드백
- 조명 상태 표시 (너무 어두움/밝음)
- 흔들림 경고

#### 3.2.2 자동 스캔 카운트다운 개선
**문제**: 현재 5프레임 안정화 후 자동 촬영이 사용자에게 불명확

**개선안**:
- 시각적 카운트다운 표시 (3, 2, 1)
- 진동/소리 피드백 옵션
- 자동 스캔 민감도 조절 설정

#### 3.2.3 캡처 피드백 강화
**개선안**:
- 촬영 시 플래시 효과
- 셔터 소리 옵션
- 촬영 성공/실패 토스트 메시지

---

### 3.3 접근성 개선

**세부 사항**:
- 다크/라이트 테마 지원
- 글꼴 크기 조절
- 고대비 모드
- VoiceOver/TalkBack 지원 강화
- 색맹 모드 지원

---

### 3.4 홈 화면 UX 개선

#### 3.4.1 빈 상태 개선
**현재**: 단순 텍스트만 표시
**개선안**:
- 애니메이션 추가
- "첫 스캔 시작하기" 버튼 강조
- 샘플 문서 미리보기

#### 3.4.2 파일 미리보기 개선
**개선안**:
- 썸네일 품질 향상
- PDF 첫 페이지 미리보기
- 파일 정보 표시 (크기, 날짜)
- 멀티 선택 모드

#### 3.4.3 스와이프 액션
**개선안**:
- 좌측 스와이프: 삭제
- 우측 스와이프: 공유
- 길게 누르기: 다중 선택

---

### 3.5 설정 화면 구현
**목표**: 사용자가 앱 동작을 커스터마이즈할 수 있게 함

**설정 항목**:
```
설정
├── 스캔 설정
│   ├── 기본 해상도
│   ├── 자동 스캔 민감도
│   ├── 기본 필터
│   └── 음성 명령 언어
├── 저장 설정
│   ├── 기본 저장 위치
│   ├── 파일명 형식
│   ├── 자동 백업
│   └── 이미지 품질
├── 표시 설정
│   ├── 테마 (다크/라이트/자동)
│   ├── 그리드 크기
│   └── 정렬 순서
└── 정보
    ├── 앱 버전
    ├── 오픈소스 라이선스
    └── 도움말
```

## 4. 버그 수정 및 안정성 개선

### 4.1 메모리 관리
**문제점**: OpenCV Mat 객체 및 이미지 처리 시 메모리 누수 가능성

**개선안**:
- [`document_scanner_service.dart`](wiscaner_app/lib/core/services/document_scanner_service.dart)에서 모든 Mat 객체 dispose 확인
- 이미지 처리 후 명시적 메모리 해제
- 대용량 이미지 처리 시 메모리 모니터링

---

### 4.2 카메라 권한 처리
**문제점**: 권한 거부 시 적절한 안내 부족

**개선안**:
- 권한 요청 다이얼로그 개선
- 권한 거부 시 설정 이동 가이드
- 카메라 사용 불가 시 대체 옵션 제공 (갤러리 선택)

---

### 4.3 예외 처리 강화
**문제점**: 일부 예외가 사용자에게 전달되지 않음

**개선안**:
- 모든 async 함수에 try-catch 추가
- 에러 로깅 시스템 구축
- 사용자에게 친절한 에러 메시지 표시

---

### 4.4 이미지 처리 성능 최적화
**문제점**: 고해상도 이미지 처리 시 지연 발생 가능

**개선안**:
- 이미지 크기 자동 조정 옵션
- 백그라운드 Isolate 처리 강화
- 진행률 표시기 추가

---

### 4.5 플랫폼별 호환성
**문제점**: Windows/Android 간 동작 차이

**개선안**:
- 플랫폼별 카메라 초기화 로직 분리
- Windows 카메라 설정 최적화
- Android 백그라운드 처리 개선

---

### 4.6 파일 저장 안정성
**문제점**: 저장 실패 시 사용자 데이터 손실 가능성

**개선안**:
- 임시 저장 후 최종 저장 방식
- 저장 실패 시 재시도 로직
- 자동 복구 기능
